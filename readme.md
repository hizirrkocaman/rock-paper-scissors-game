# Rock Paper Scissors Game

A simple Rock–Paper–Scissors game built with **Vanilla JavaScript**, featuring **keyboard controls**, **auto play**, and **persistent score** using `localStorage`.

## Demo

- Live Demo: https://hizirrkocaman.github.io/rock-paper-scissors-game

## Features

- Play with buttons: **Rock / Paper / Scissors**
- Keyboard controls:
  - **R** → Rock
  - **P** → Paper
  - **S** → Scissors
- **Auto play** mode (plays automatically every second)
  - While auto play is running, manual controls are disabled (buttons + keyboard)
- **Persistent score** (wins / losses / ties) saved in `localStorage`
- **Reset score** button

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
